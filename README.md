# Projeto Eshet Chayil

Site institucional do **Projeto Eshet Chayil (Empreendendo e Fortalecendo Mulheres Átipicas)**.

## Sobre o projeto
O site apresenta a proposta social do Projeto Eshet Chayil, com foco em:

- acolhimento de mulheres atípicas;
- apoio à saúde mental;
- encontros mensais de fortalecimento;
- oficinas práticas e incentivo à autonomia;
- rede de apoio social e comunitária;
- apresentação da iniciativa Anjo Eshet Chayil para apoiadores e parceiros.

## Estrutura principal
- **Home** com mensagem institucional e chamada para participação;
- **Sobre o projeto**;
- **Como ajudamos**;
- **Impacto social**;
- **Seja um Anjo Eshet Chayil**;
- **FAQ**;
- **Seção de cadastro / apoio**.

## Identidade aplicada
- Nome curto em áreas compactas: **Projeto Eshet Chayil**
- Nome completo em áreas institucionais: **Projeto Eshet Chayil (Empreendendo e Fortalecendo Mulheres Átipicas)**
- Linguagem acolhedora, profissional e inspiradora
- Foco social, inclusivo e humano

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo desta versão
Esta versão foi revisada para remover referências antigas e alinhar o site à proposta oficial do projeto social, fortalecendo a apresentação para participantes, apoiadores, parceiros e instituições.
