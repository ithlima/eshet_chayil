# Projeto Eshet Chayil (Empreendendo e Fortalecendo Mulheres Átipicas)

Este site foi atualizado para refletir integralmente a nova identidade institucional do projeto social **Projeto Eshet Chayil (Empreendendo e Fortalecendo Mulheres Átipicas)**.

## Atualizações aplicadas

- substituição completa do nome antigo por **Projeto Eshet Chayil**;
- atualização de `<title>`, metadados SEO e descrições institucionais;
- revisão da marca textual da navbar, hero, FAQ, rodapé e textos de apresentação;
- reescrita do conteúdo institucional para reforçar acolhimento, inclusão, fortalecimento e empreendedorismo voltado a mulheres atípicas;
- manutenção da estrutura do site, links internos, âncoras e comportamento em JavaScript.

## Identidade adotada

- **Nome completo:** Projeto Eshet Chayil (Empreendendo e Fortalecendo Mulheres Átipicas)
- **Nome reduzido:** Projeto Eshet Chayil
- **Proposta:** acolhimento, apoio, fortalecimento, inclusão, valorização e empreendedorismo para mulheres atípicas.

## Estrutura principal

- `index.html`: página principal com toda a identidade textual atualizada;
- `assets/css/`: estilos visuais do site;
- `assets/js/script_inicial.js`: interações da interface;
- `assets/imgs/` e `assets/videos/`: mídia usada no projeto.

## Observação

Foi feita uma revisão textual completa do conteúdo disponível no projeto entregue. Nenhuma rota, âncora ou funcionalidade principal foi alterada.
